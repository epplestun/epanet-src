excluded

excluded: Boolean|Function — Indicate whether or not the field is excluded.

To exclude a set of fields, you should use the excluded option for form.

{% highlight javascript %} // Use Boolean value excluded: true or false // Use the return value of a callback function excluded: function($field, validator) { // $field is the field element // validator is the plugin instance return true; // or false } {% endhighlight %}
