formvalidation.io
=================

This repository contains the source of [formvalidation.io](http://formvalidation.io)

## Installing

```
$ gem install jekyll
$ gem install bundler
```

Go to the root directory and install the required plugins with

```
$ bundle install
```

## Building

```
$ jekyll build
$ jekyll build --watch
```

Build and serve the docs in local with

```
$ jekyll serve
```
