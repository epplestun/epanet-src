removeField

removeField(field*): FormValidation — Remove given field

Parameter Type Description
field String|jQuery The field name or field element

By triggering the removed.field.fv event, you can perform additional task after removing given field:

{% highlight javascript %} $(document).ready(function() { $(form) .formValidation(options) .on('removed.field.fv', function(e, data) { // $(e.target) --> The form instance // $(e.target).data('formValidation') // --> The FormValidation instance // data.field --> The field name // data.element --> The new field element // Do something ... }); }); {% endhighlight %}
